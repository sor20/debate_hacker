# 辩论修改器 - Debate Hacker

一个功能强大的辩论辅助工具，提供多种实用功能帮助你在辩论比赛中取得优势。

## 🚀 功能特性

### 1. 时间调整模块
- **陈词时间**：支持+10秒和+999秒快速调整
- **质询时间**：支持+10秒和+999秒快速调整
- 实时显示当前时间状态

### 2. 评委好感度修改
- 提供直观的滑块控制
- 范围从0到100，实时调整评委对你的好感度

### 3. 票型透视功能
- **三位评委**：印象票、环节票、决胜票（共9票）
- **实时计算**：自动统计正方和反方得票数，确保总和为9票
- **票数规则**：
  - 印象票：正/反/平（1/1/0.5票/方）
  - 环节票：正/反/平（1/1/0.5票/方）
  - 决胜票：正/反/平（1/1/0.5票/方）
- 自动显示最终赢家

### 4. 无敌模式
- **无敌申论**：开启后在申论环节获得无敌状态
- **无敌结辩**：开启后在结辩环节获得无敌状态
- **无敌质询**：开启后在质询环节获得无敌状态
- **一键全选**：快速开启所有无敌模式

### 5. 履历修改器
- **国际赛**：三类履历数量显示
- **无限制级**：三类履历数量显示
- **限制级**：三类履历数量显示
- **佳辩次数**：一键增加最佳辩手次数
- **全程最佳**：一键增加全程最佳辩手次数
- 实时更新履历统计数据

## 📋 安装要求

### 系统要求
- Windows 7/8/10/11

### 依赖安装
```bash
pip install pyqt5
```

## 🎮 使用方法

### 直接运行Python脚本
```bash
python debate_hacker.py
```

### 生成可执行文件（EXE）
1. 安装PyInstaller
```bash
pip install pyinstaller
```

2. 生成EXE文件（包含嵌入式音乐）
```bash
pyinstaller debate_hacker.spec
```

3. 在`dist`文件夹中找到生成的`debate_hacker.exe`文件

## 🎨 界面说明

### 顶部栏
- **辩手选择器**：选择正方或反方，输入辩手姓名
- **时间显示**：显示当前陈词和质询时间

### 左侧面板
- **时间调整**：快速修改陈词和质询时间
- **评委好感度**：调整评委对你的好感度

### 中间面板
- **票型透视**：查看和修改三位评委的投票情况
- **实时统计**：显示当前得票和最终结果

### 右侧面板
- **无敌模式**：开启各环节的无敌状态
- **履历修改器**：调整各类比赛履历数量

## 📊 票型计算规则

总票数：3位评委 × 3种票型 = 9票

### 投票逻辑
- **正**：为正方加1票
- **反**：为反方加1票  
- **平**：双方各加0.5票

### 示例
- 评委1：印象票（正）、环节票（正）、决胜票（平）→ 正方2.5票
- 评委2：印象票（反）、环节票（平）、决胜票（正）→ 正方1.5票，反方1票
- 评委3：印象票（平）、环节票（平）、决胜票（平）→ 双方各1.5票
- **总计**：正方5.5票，反方2.5票 → 正方获胜
